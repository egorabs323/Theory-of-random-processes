# Расчетно-Графическая Работа: Анализ Временных Рядов

## Описание проекта

Этот проект содержит полную реализацию расчетно-графической работы по анализу временных рядов на Python. Работа охватывает все аспекты, указанные в задании:

- Визуальный анализ временного ряда
- Анализ выбросов (3 метода: IQR, MAD, Z-score)
- Построение и интерпретация коррелограмм (АКФ и ЧАКФ)
- Проверка стационарности (ADF и KPSS тесты)
- Разработка и обучение 3+ моделей:
  - **ARIMA** - авторегрессионная интегрированная модель скользящего среднего
  - **Exponential Smoothing** - экспоненциальное сглаживание Хольта-Винтерса
  - **TBATS** - продвинутая модель для сложных сезонных паттернов
- Оценка качества моделей (MAE, MSE, MAPE, RMSE)
- Анализ остатков и диагностика моделей
- Исследование влияния параметров на качество
- Анализ зависимости точности от горизонта прогнозирования
- Генерация полного HTML отчета

## Метрики качества

Для оценки качества моделей используются следующие метрики:

- **MAE (Mean Absolute Error)** - средняя абсолютная ошибка
  - MAE = (1/n) * Σ|y_t - ŷ_t|
  - Интерпретация: среднее абсолютное отклонение прогноза от фактических значений

- **MSE (Mean Squared Error)** - среднеквадратичная ошибка
  - MSE = (1/n) * Σ(y_t - ŷ_t)²
  - Интерпретация: средний квадрат ошибки (штрафует большие ошибки)

- **RMSE (Root Mean Squared Error)** - корень из MSE
  - RMSE = √MSE
  - Интерпретация: в тех же единицах, что исходные данные

- **MAPE (Mean Absolute Percentage Error)** - средняя абсолютная процентная ошибка
  - MAPE = (100/n) * Σ|y_t - ŷ_t| / |y_t|
  - Интерпретация: процент ошибки относительно фактических значений

## Анализ остатков

Для каждой модели анализируются:

1. **График остатков** - проверка случайности
2. **Гистограмма остатков** - проверка нормальности распределения
3. **АКФ остатков** - проверка автокорреляции
4. **Q-Q диаграмма** - проверка соответствия нормальному распределению

## Методология

### 1. Выявление выбросов

**Метод IQR (Interquartile Range)**
- Выявляет точки вне диапазона [Q1 - 1.5*IQR, Q3 + 1.5*IQR]

**Метод MAD (Median Absolute Deviation)**
- Использует модифицированный z-score на основе медианы
- Более устойчив к экстремальным значениям

**Метод Z-score**
- Выявляет значения, отклоняющиеся на более чем 3σ от среднего

### 2. Проверка стационарности

**Тест Дики-Фуллера (ADF)**
- H₀: ряд имеет единичный корень (нестационарен)
- p-value < 0.05 → отвергаем H₀ → ряд стационарен

**Тест KPSS**
- H₀: ряд стационарен
- p-value > 0.05 → не отвергаем H₀ → ряд стационарен

### 3. Коррелограммы

**АКФ (Autocorrelation Function)**
- Показывает корреляцию ряда с самим собой при различных лагах
- Медленное убывание указывает на нестационарность
- Периодические всплески указывают на сезонность

**ЧАКФ (Partial Autocorrelation Function)**
- Показывает корреляцию после исключения влияния промежуточных лагов
- Помогает определить порядок авторегрессии (p)

## Интерпретация результатов

### Выбор лучшей модели

1. Первичный критерий: **наименьшее значение MAE или RMSE**
2. Вторичный критерий: **наименьшее значение MAPE**
3. Третичный критерий: **качество остатков**

### Анализ остатков

✓ Хорошие остатки:
- График остатков - случайные колебания вокруг нуля
- Гистограмма - близка к нормальному распределению
- АКФ остатков - все корреляции в доверительных полосах
- Q-Q диаграмма - точки лежат на прямой

✗ Проблемы:
- Систематическое смещение - недостаточный тренд/сезонность
- Гетероскедастичность - изменение дисперсии со временем
- Автокорреляция остатков - неправильный выбор параметров
- Ненормальность - возможны выбросы или неправильная трансформация

## Troubleshooting

### Ошибка: "ModuleNotFoundError: No module named 'statsmodels'"
```bash
pip install statsmodels
```

### Ошибка: "ImportError: cannot import name 'TBATS'"
```bash
pip install --upgrade tbats
```

### Ошибка: "FileNotFoundError: wordstat_dynamic.xlsx"
- Убедитесь, что файл находится в текущей директории
- Проверьте правильность названия файла

### Графики не сохраняются
- Проверьте наличие папки `figures/`
- Убедитесь в наличии прав на запись в текущей директории
